规则信息配置

| 字段            | 类型      | 描述                                                                 |
| --------------- | --------- | -------------------------------------------------------------------- |
| `categoryID`    | `string`  | 分类的唯一标识符                 |
| `ruleTitle`     | `string`  | 标题的本地化键。通常格式为 `#STR_...`，也可以直接写名称 |
| `ruleIcon`      | `string`  | 图标路径。支持 `.edds` 格式。                  |
| `rulesList`     | `array string` | 包含规则内容的字符串数组。    |
