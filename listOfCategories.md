分类配置

| 字段            | 类型      | 描述                                                                 |
| --------------- | --------- | -------------------------------------------------------------------- |
| `showCategory`  | `int`     | 是否显示该分类（1 — 显示，0 — 隐藏）。                                |
| `categoryID`    | `string`  | 分类的唯一标识符，用于代码中，禁止修改！                              |
| `categoryTitle` | `string`  | 分类名称的本地化键。通常形式为 `#STR_...`，也可以直接写翻译后的名称     |
| `categoryIcon`  | `string`  | 分类图标的路径。支持 `.edds` 格式。                                   |

在配置中添加分类只会让它显示在界面上，但点击时不会发生任何操作。  
你只能修改分类的名称、显示状态和图标。
